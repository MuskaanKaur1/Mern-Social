# MERN Social 

#### What you need to run this code
1. Node (13.12.0)
2. NPM (6.14.4) or Yarn (1.22.4)
3. MongoDB (4.2.0)

####  How to run this code
1. Make sure MongoDB is running on your system 
2. Clone this repository
3. Open command line in the cloned folder,
   - To install dependencies, run ```  npm install  ``` or ``` yarn ```
   - To run the application for development, run ```  npm run development  ``` or ``` yarn development ```
4. Open [localhost:3000](http://localhost:3000/) in the browser
---- 

A full-stack social media platform built with the MERN stack, featuring user posts, likes, comments, and secure authentication.

* Features

1. User Authentication & Authorization using JWT
2. Create, Read, Update, Delete (CRUD) Posts
3. Like & Comment on Posts
4. User Profiles
5. Protected Routes for authenticated users
6. Scalable Backend APIs with Express.js
7. Structured MongoDB Schemas using Mongoose
